Completed the Best time to buy and sell stock problem.
Used kind of a two pointer solution that points to minimum value amongst the values already encountered and one that goes with the latest value in the array according to the loop.
Firstly solved it in O(n) time complexity and O(1) space complexity which required 6ms time.
Later optimized it to return 3ms and finally 0ms time and 97.39 MB space which is still the same time and space complexity with just a little modification like taking out unused variables and such.
The final algorithm takes O(n) time complexity and O(1) space complexity which in leetcode gives us 0ms of time and 97.39 MB of space requirement.
